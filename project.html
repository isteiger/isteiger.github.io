Coding Helps the World

Overview

How does coding help the world?

Coding helps the world by letting us express things we have made and lets us create things that can help others. Websites are one thing created by coding Websites can share information and help provide access to things such as healthcare and calendars. Apps are another thing created by coding. Apps can connect people and help humans with tasks, such as automating when a dryer runs.

What are some examples of how coding has helped the world?

  • Facebook

    Facebook has connected many people throughout the world. People can connect with Facebook with people from everywhere, which has helped the world.

  • Google Classroom

    Google Classroom helps us learn remotely, which has helped the world a lot. Without it we would not learning very effictively from home.

  • Zoom

    Zoom and services like Zoom have helped us communicate, have live instruction, and meetings while still being at home.

How can I get into coding websites?

  1. Go over to W3Schools HTML and start learning there.
  2. After learning HTML, you will want to learn CSS to make the sites you make look better.
  3. After that, you will want to learn how to make your site dynamic on the client-side with JavaScript.
  4. Finally, learn PHP or another server-side language.
